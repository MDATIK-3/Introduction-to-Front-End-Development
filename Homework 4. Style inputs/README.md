# Homework 4: Style Inputs

This homework assignment focuses on styling HTML input elements using CSS. The goal is to create visually appealing and accessible form inputs with different sizes and color themes.

## Objectives

- Create input fields with three different sizes: small, medium, and large
- Implement two color themes: primary (green) and secondary (gray)
- Ensure proper focus states and accessibility
- Use CSS classes for modular styling

## Features

- **Sizes**: Small (12px font, compact padding), Medium (16px font, standard padding), Large (20px font, generous padding)
- **Themes**: 
  - Primary: Green background with white text
  - Secondary: Light gray background with dark text
- **Accessibility**: Focus indicators with outline, proper contrast ratios
- **Responsive**: Fixed width inputs with smooth transitions

## Files

- `index.html`: The HTML structure containing a form with three styled input fields
- `styles.css`: The CSS styles defining the input appearance, sizes, and themes

## How to View

Open `index.html` in a web browser to see the styled form inputs.
